Visual CertExam Manager & A+ VCE

Panduan install dan menggunakan Visual CertExam Manager untuk membuka file VCE

1

Extract Visual CertExam Suite

Download dan extract file Visual CertExam Suite. Setelah di-extract, kamu akan melihat isi folder seperti berikut:

Isi folder: Registration Key, APK files, instruction.txt, dan setup.exe
2

Install Visual CertExam Suite

Klik dua kali pada file visual_certexam_suite_setup.exe untuk memulai instalasi:

Jalankan file setup.exe untuk install

Ikuti wizard instalasi: Next ‚Üí Accept ‚Üí Install ‚Üí Finish

Penting!

Setelah instalasi selesai, JANGAN langsung buka aplikasi! Lanjut ke langkah berikutnya dulu.

3

Copy Registration Key

Buka folder Registration Key dan copy file yang ada di dalamnya:

Copy file Visual_CertExam_Suite_1.9.987_FiX.reg

Paste file tersebut ke folder instalasi:

C:\Program Files (x86)\Visual CertExam Suite
Paste Registration Key ke folder instalasi
4

Aktivasi Software

Klik dua kali pada file Registration Key yang sudah di-paste, lalu klik OK pada dialog yang muncul.

Berhasil!

Software Visual CertExam Suite sudah terdaftar dan siap digunakan.

5

Membuka File VCE

Untuk membuka file VCE di HP Android, gunakan aplikasi A+ VCE Silver:

Klik File untuk membuka file VCE
Tips

Uncheck opsi "Shuffle questions and choices" jika ingin soal tampil sesuai urutan aslinya.
