Pre Config SD-WAN Devices

Panduan manual provisioning dan Day 0 configuration untuk berbagai device Cisco SD-WAN

Tentang Pre-Configuration

Pre-configuration diperlukan untuk onboarding device ke SD-WAN fabric secara manual. Gunakan panduan ini jika:

  • Device tidak memiliki akses internet untuk PnP
  • Menggunakan on-premise vManage
  • Environment lab atau testing yang terisolasi
  • Kebijakan keamanan tidak mengizinkan cloud PnP
