Cisco ISR1100-4GLTEGB Pre-Configuration

Panduan konfigurasi manual untuk onboarding ISR1100-4GLTEGB ke SD-WAN fabric dengan LTE connectivity

Informasi Device

Cisco ISR1100-4GLTEGB adalah router dengan kapabilitas SD-WAN dan konektivitas 4G LTE. Device ini cocok untuk deployment di lokasi remote atau sebagai backup WAN connection.

Parameter Value Keterangan
Model ISR1100-4GLTEGB Cisco ISR 1100 Series dengan 4G LTE
System IP 10.255.229.228 IP address unik untuk identifikasi di SD-WAN overlay
Site ID 203420027 Identifier unik untuk site location
Organization telkom-mt1-diskominfo-sukabumi Nama organisasi device
SP Organization telkom-mt1 Service Provider organization
Console Baud Rate 9600 Speed koneksi console
vBond Address 36.94.252.25:12346 vBond orchestrator address

System Configuration

Basic System Settings

Konfigurasi dasar system yang harus diset pada device sebelum join ke SD-WAN fabric:

system configuration 
system
 system-ip             10.255.229.228
 site-id               203420027
 sp-organization-name  telkom-mt1
 organization-name     telkom-mt1-diskominfo-sukabumi
 console-baud-rate     9600
 vbond 36.94.252.25 port 12346

Penting: Pastikan system-ip dan site-id sudah diassign oleh vManage administrator sebelum melakukan konfigurasi ini. Nilai harus unik di seluruh SD-WAN fabric.

Tunnel Interface Configuration

WAN Interface (GE0/0)

Konfigurasi tunnel interface untuk transport WAN. Interface ini akan digunakan untuk membentuk overlay tunnel ke SD-WAN controllers.

tunnel interface config 
vedge# config
vedge(config)# interface-ge0/0
vedge(config-interface-ge0/0)# tunnel-interface
vedge(config-tunnel-interface)# color biz-internet
vedge(config-tunnel-interface)# commit
Commit complete.
vedge(config-tunnel-interface)#

Commit Successful: Konfigurasi tunnel interface berhasil disimpan. Device akan mulai membentuk control dan data plane connections ke SD-WAN controllers.

Parameter Value Keterangan
Interface GE0/0 Gigabit Ethernet port 0/0
Tunnel Interface enabled Enable tunnel pada interface ini
Color biz-internet Business Internet transport color

Langkah-langkah Konfigurasi

Akses Console Device
Hubungkan ke console port dengan baud rate 9600. Gunakan terminal emulator seperti PuTTY atau SecureCRT.
Masuk ke Configuration Mode
Ketik config untuk masuk ke configuration mode. Prompt akan berubah menjadi vedge(config)#
Konfigurasi System Parameters
Set system-ip, site-id, organization-name, dan vbond address sesuai dengan nilai yang diberikan oleh administrator.
Konfigurasi Tunnel Interface
Masuk ke interface GE0/0, enable tunnel-interface, dan set color ke biz-internet.
Commit Configuration
Ketik commit untuk menyimpan konfigurasi. Tunggu sampai muncul pesan "Commit complete."
Verifikasi Koneksi
Cek status control connection dengan perintah show control connections. Pastikan semua controller dalam state "up".

Verification Commands

Check Control Connections

show control connections 
vedge# show control connections

Check Interface Status

show interface 
vedge# show interface ge0/0

Check System Status

show system status 
vedge# show system status
