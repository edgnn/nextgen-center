Clear CRC Error

Tujuan: Mereset semua counter error (termasuk CRC Error) pada interface ke nol. Biasanya dilakukan setelah perbaikan fisik (cleaning konektor, ganti kabel/SFP) untuk memastikan error tidak muncul lagi setelah perbaikan.

Perhatian!

Perintah ini akan mereset counter error pada interface. Pastikan sudah mencatat data error sebelum melakukan clear.

CLI — Clear Interface Counters 
RTR5-KLM-CTRL-CNBC-01> enable
RTR5-KLM-CTRL-CNBC-01#
RTR5-KLM-CTRL-CNBC-01# clear switch R5.KLM.LEAF-CNDC.01 interface ethernet35 counters
🔄

Reset Interface

Tujuan: Mengembalikan interface yang berstatus error-disabled ke kondisi normal. Interface bisa masuk status error-disabled karena BPDU guard, storm control, atau error yang berlebihan. Perintah ini akan mengaktifkan kembali interface tersebut.

Perhatian!

Perintah ini akan mereset interface yang dalam status error-disabled. Pastikan sudah mengecek penyebab error sebelum melakukan reset.

CLI — Clear Error-Disabled Interface 
RTR5-KLM-CTRL-CNBC-01# clear error-disabled switch R5.KLM.LEAF-CNDC.01 interface ethernet35
